## 核心定义

### 产品类型

| 类型 | 说明 | 代表场景 |
|------|------|----------|
| **场景分期** | 在特定消费场景分期 | 医美、教育、装修 |
| **通用分期** | 不限制消费场景 | 花呗分期、信用卡分期 |
| **现金分期** | 发放现金后分期还款 | 信用卡现金分期 |

### 与信用卡的区别

| 维度 | 消费分期 | 信用卡 |
|------|----------|--------|
| 额度 | 单次额度 | 循环额度 |
| 用途 | 指定场景 | 任意消费 |
| 免息期 | 无 | 有 |
| 利率 | 固定费率 | 循环利率 |

## 商业模式

### 三方模式

```
┌─────────────────────────────────────────────────────────────┐
│  消费者 ←→ 场景方（医美/教育/电商）                        │
│              ↓                                              │
│         资金方（银行/消金）                                  │
└─────────────────────────────────────────────────────────────┘
```

### 收入结构

| 收入来源 | 收取对象 |
|----------|----------|
| **分期手续费** | 消费者 |
| **服务费** | 场景方 |
| **担保费** | 担保公司 |

### 担保模式

| 模式 | 说明 |
|------|------|
| 场景方担保 | 场景方承担兜底 |
| 担保公司 | 第三方担保 |
| 保险公司 | 信保模式 |
| 资产穿透 | ABS出表 |

## 场景分析

### 主要场景

| 场景 | 市场规模 | 特点 |
|------|----------|------|
| **医美分期** | 千亿级 | 客单价高，决策周期长 |
| **教育分期** | 千亿级 | 政策敏感，场景方重要 |
| **3C数码** | 万亿级 | 标准化，电商主导 |
| **家电分期** | 千亿级 | 低频高额 |
| **装修分期** | 万亿级 | 重线下，周期长 |

### 医美分期特点

| 维度 | 说明 |
|------|------|
| 额度 | 几千-几十万 |
| 期数 | 3/6/12/24期 |
| 客户 | 女性为主，25-40岁 |
| 风险 | 退贷纠纷、骗贷 |

### 教育分期特点

| 维度 | 说明 |
|------|------|
| 额度 | 几千-几万 |
| 期数 | 6/12/24/36期 |
| 客户 | 年轻人职业技能培训 |
| 风险 | 培训机构跑路 |
