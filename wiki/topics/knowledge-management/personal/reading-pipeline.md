## 步骤详解

### 1. 阅读与标注

| 方式 | 工具 | 适用场景 |
|------|------|----------|
| **电子书** | Kindle、微信读书 | 小说、商业书 |
| **PDF** | PDF Reader、Marginnote | 论文、报告 |
| **网页** | Readwise Reader、Pocket | 文章、博客 |
| **音频** | 播客、有声书 | 通勤时间 |

### 2. 闪念笔记

**目的**: 快速捕获阅读时的想法

```
格式：Fleeting Note
├── 来源：（书名/文章名）
├── 想法：（你的即时反应）
└── 引用：（原文摘录，可选）
```

### 3. 文献笔记

**目的**: 记录阅读内容的精华

```
格式：Literature Note
├── 来源信息（书名、作者、页码）
├── 核心观点（1-3个）
├── 精彩摘录（原文）
└── 我的理解（用自己话重述）
```

### 4. 永久笔记

**目的**: 把知识变成自己的

```
格式：Permanent Note
├── 核心想法（一个笔记一个想法）
├── 链接到相关笔记
├── 思考：这个知识如何应用到我的工作/生活
└── 标签：#主题 #领域
```

### 5. 建立链接

**原则**:
- 新笔记 → 旧笔记：找到关联
- 类似观点 → 链接在一起
- 相反观点 → 也链接，标记分歧

### 6. 实践应用

| 应用方式 | 说明 |
|----------|------|
| **写作** | 用自己的话写出来 |
| **分享** | 讲给别人听 |
| **实践** | 应用到工作/生活 |
| **教学** | 教别人 |

## 不同类型阅读的处理

### 碎片化阅读（文章、推文）

```
快速阅读
    │
    ▼
判断价值
    │
    ├── 值得深读 → 存入Readwise/稍后读
    │         │
    │         ▼
    │     深入阅读 → 文献笔记 → 永久笔记
    │
    └── 不值得 → 简单记录想法 → 归档
```

### 系统化阅读（书籍、论文）

```
确定阅读目标
    │
    ▼
快速预览（目录、序言、结论）
    │
    ▼
深度阅读（带着问题）
    │
    ├── 做标注/笔记
    │
    └── 章节结束后写文献笔记
    │
    ▼
阅读完毕写总结笔记
    │
    ▼
提取关键概念写永久笔记
    │
    ▼
建立链接 → 形成知识网络
```

## 关键原则

### 1. 读完要消化

> 读完一本书不叫学习，把书的内容讲出来才叫学习。

### 2. 笔记要转化

> 把作者的话变成自己的话，才算真正理解了。

### 3. 链接要主动

> 不要等链接自己出现，要主动去找关联。

### 4. 输出要持续

> 写作是最好的学习，输出倒逼输入。
