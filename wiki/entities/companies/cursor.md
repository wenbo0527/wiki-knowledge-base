## 技术特点

### 1. AI原生架构

```
传统IDE + AI插件
    │
    └── AI是附加层，在编辑器上运行

Cursor
    │
    └── AI是核心，编辑器围绕AI构建
    │
    ├── AI理解你的整个代码库
    ├── AI可以编辑多个文件
    └── AI可以运行命令和搜索
```

## Vibe Coding与Cursor

### Cursor是Vibe Coding的理想工具

| Vibe Coding要素 | Cursor支持 |
|-----------------|------------|
| **自然语言编程** | ✅ Ctrl+L对话 |
| **上下文固定** | ✅ 整个代码库理解 |
| **快速迭代** | ✅ 实时预览 |
| **多文件生成** | ✅ Composer |

### Cursor使用技巧

1. **明确角色** — 给Cursor一个身份（如"资深Python后端"）
2. **分步执行** — 不要一次让Cursor做太多
3. **利用补全** — Cursor的代码补全比Copilot更快
4. **善用Composer** — 复杂功能先用Composer生成框架
