## 核心定义

### 业务模式

| 模式 | 定义 | 平台角色 |
|------|------|----------|
| **助贷** | 平台提供流量+风控，银行出资金 | 流量+风控 |
| **联合贷款** | 多家机构按比例出资+风控 | 联合出资 |
| **导流** | 平台仅提供客户信息 | 仅获客 |
| **担保贷** | 平台提供担保，银行出资 | 担保兜底 |

### 助贷 vs 联合贷款

| 维度 | 助贷 | 联合贷款 |
|------|------|----------|
| 出资比例 | 银行100% | 按比例 |
| 风控 | 平台输出 | 联合风控 |
| 收益分配 | 固定服务费 | 按比例分成 |
| 风险承担 | 平台担保/兜底 | 按比例承担 |

## 合作模式

### 典型合作结构

```
┌─────────────────────────────────────────────────────────────┐
│  场景/流量平台                                              │
│  电商、线下场景、社交平台                                    │
└─────────────────────────────────────────────────────────────┘
         ↓                                    ↑
┌─────────────────────────────────────────────────────────────┐
│  助贷/联合贷款平台                                          │
│  获客 + 风控 + 催收 + 担保                                  │

         ↓
```

## 监管政策

### 重要监管文件

| 政策 | 核心要求 |
|------|----------|
| 141号文 | 银行不得为无资质机构提供资金 |
| 银保监办发2019 | 联合贷款出资比例要求 |
| 网络小贷新规 | 跨省经营限制 |
| 融资担保新规 | 担保公司杠杆倍数 |

### 合规要点

| 要点 | 说明 |
|------|------|
| 资质要求 | 助贷机构需有担保证 |
| 杠杆限制 | 融资担保公司杠杆≤10x |
| 利率限制 | 综合费率36%以内 |
| 资金不得外包 | 核心风控不得外包 |
