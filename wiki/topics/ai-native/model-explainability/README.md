## 专题定位
本专题聚焦**金融行业尤其是消费金融场景下的模型可解释性落地**，覆盖监管要求、方法论、工具链、实战案例全链路，解决黑盒模型在金融场景落地的合规性问题，满足"模型可解释、决策可追溯、拒贷可说明"的监管要求。
### 核心价值
1. **合规要求**: 满足巴塞尔协议、银保监会、人行等监管机构对金融模型可解释性的强制要求，避免合规处罚
2. **业务信任**: 模型决策可解释，提升业务方、用户对模型的信任度，减少用户申诉
3. **风险可控**: 可解释性帮助定位模型缺陷，避免模型出现系统性风险
4. **体验提升**: 清晰的拒贷理由提升用户体验，减少投诉
