## 💡 深度分析

### 1. Product-Manager-Skills ⭐ 3.5k

**定位**：教AI Agent做产品管理

**核心理念**：
> "Battle-tested PM frameworks that teach both you and your AI agents how to do product management work at a professional level."

**47个Skill分类**：
| 类别 | 数量 | 代表Skill |
|------|------|-----------|

| PRD开发 | 3 | PRD Development, Press Release |

### 2. SuperDesign ⭐ 6.3k

**定位**：第一个开源AI设计Agent

**核心功能**：
| 功能 | 说明 |
|------|------|
| Product Mock | 一句话生成完整UI |
| UI Components | 生成可复用组件 |
| Wireframes | 低保真线框图 |
| Fork & Iterate | 复制并迭代 |

**PM使用场景**：
```
需求描述 → AI生成10种方案 → 选择最优 → 微调 → 交付开发
```

**优势**：
> "Why design one option when you can explore ten?"

---

### 3. Mission Control (Autensa) ⭐ 1.9k

**定位**：世界第一个Autonomous Product Engine

**Slogan**：
> "Your products improve themselves — 24/7 — while you sleep."

**工作流**：
```
Research → Ideation → Swipe → Build → Test → Review → PR (全自动)
```

**技术栈**：
- Next.js 14
- TypeScript
- SQLite
- OpenClaw Agent
