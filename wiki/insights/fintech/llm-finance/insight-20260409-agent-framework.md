## 核心洞察

### 框架选型矩阵

| 框架 | 定位 | 优势 | 劣势 | 场景 |
|------|------|------|------|------|
| **LangChain** | 全能型 | 生态丰富 | 复杂 | 快速原型 |
| **LangGraph** | 可控型 | 精细控制 | 编码量大 | 生产系统 |
| **AutoGen** | 多Agent | 微软官方 | 文档欠缺 | 多Agent协作 |
| **CrewAI** | 多Agent | 概念清晰 | 定制性弱 | 团队Agent |

## 多Agent架构

### 分层模式（推荐）

```
Manager → 任务分解、结果汇聚
    ↓
Worker1 / Worker2 / Worker3 → 各司其职
```

### 常见问题

| 问题 | 解决 |
|------|------|
| 群聊死锁 | 超时机制+强制输出 |
| 信息丢失 | 结构化消息格式 |
| 状态不一致 | 共享状态存储 |
