## 原文核心信息

**标题**: 第005篇｜PM Skills 完整教程：产品经理最实用的 65 个 Skill 一次讲清

**核心内容**:
- 8大模块：发现与用户研究、假设与优先级、规划与交付、分析与指标、GTM与增长、策略与商业模型、日常工具箱、附录
- 65个实用Skills（从用户访谈、机会评估、Gherkin需求到A/B测试、北极星指标）
- GitHub开源项目：8.7k stars，活跃维护

**上手主线**（推荐的5步流程）:
1. interview-script：先从用户研究开始
2. create-prd：把问题和方案整理成正式文档
3. prioritize-features：面对一堆需求时，决定先做什么
4. sprint-plan：把优先级落到实际迭代里
5. metrics-dashboard：建立指标闭环

**给新手的建议**:
- 不是要把 65 个 skill 都背熟，而是先知道自己现在在哪一步
- PM Skills 最适合按"研究 → 定义 → 优先级 → 交付 → 指标 → 增长 → 战略"这条主线去理解
