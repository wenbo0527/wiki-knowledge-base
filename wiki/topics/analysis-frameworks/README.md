## 框架对比

| 维度 | 何刚 | 马江博 | 蔡钰 | 万维钢 |
|------|------|--------|------|--------|
| **分析层次** | 宏观→行业→公司 | 趋势→结构→节奏 | 商业→用户→产品 | 科学→验证→迁移 |
| **核心关注** | 投资价值 | 机会识别 | 商业逻辑 | 底层规律 |
| **输出形式** | 研报 | 洞察 | 策略 | 认知升级 |
| **适用周期** | 中长期 | 中短期 | 中期 | 长期 |

## 如何选择框架

```
分析目标
    │
    ├── 投资决策 → 何刚框架（宏观+行业+公司）
    │
    ├── 机会识别 → 马江博框架（趋势+结构）
    │
    ├── 商业变现 → 蔡钰框架（逻辑+价值）
    │
    ├── 认知升级 → 万维钢框架（科学+迁移）
    │
    └── 综合分析 → 10大分析模型组合
```

## 实战应用

### 案例：分析一家AI公司

| 框架 | 分析角度 | 输出 |
|------|----------|-----|
| 何刚 | 宏观AI政策→行业增速→公司护城河 | 投资评级 |
| 马江博 | AI趋势→竞争结构→产品节奏 | 市场机会 |
| 蔡钰 | 商业模式→用户痛点→产品价值 | 商业化建议 |
| 万维钢 | 物理思维→激励分析→认知偏差 | 决策优化 |
