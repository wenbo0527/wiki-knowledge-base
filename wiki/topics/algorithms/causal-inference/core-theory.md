## 1. 潜在结果框架 (Rubin Causal Model, RCM)

### 1.1 基本定义

```
核心定义
- 处理 (Treatment, T): 我们关心的干预
- 潜在结果 (Potential Outcome, Y(t)): 如果接受处理t时的结果
- 个体处理效应 (τi): τi = Yi(1) - Yi(0)
- 平均处理效应 (ATE): E[Y(1) - Y(0)]
- 处理组平均处理效应 (ATT): E[Y(1)-Y(0)|T=1]
- 对照组平均处理效应 (ATC): E[Y(1)-Y(0)|T=0]
```

### 1.2 三大核心假设

| 假设 | 定义 | 检验方法 |
|------|------|----------|
| **1. 可忽略性 (Ignorability)** | 给定协变量X，处理分配独立于潜在结果 | 随机化实验可满足；观察性数据用倾向得分匹配 |
| **2. 正性 (Positivity)** | 对所有协变量组合，都有被分配到处理和对照组的概率 | 检查协变量重叠 |
| **3. 一致性 (Consistency)** | 如果实际接受的处理是t，那么观察到的结果等于潜在结果Y(t) | 无法检验，靠研究设计保证 |

## 2. 结构因果模型 (Structural Causal Model, SCM)

### 2.1 基本组成

```
SCM = (U, V, F, P(U))
- U: 外生变量（未观察到）
- V: 内生变量（观察到）
- F: 结构方程，决定每个变量的值
- P(U): 外生变量的概率分布
```

### 2.2 因果图 (Causal DAG)

用有向无环图表示变量间的因果关系：
- 节点：变量
- 边A→B：A直接导致B
- 没有边：没有直接因果关系
- 没有环：因果关系有方向

### 2.3 三类基本结构

| 结构 | 图示 | 特征 | 处理方法 |
|------|------|------|----------|
| **链式结构** | A→B→C | B是中介变量 | 控制B可以阻断路径 |
| **叉式结构** | A←C→B | C是混杂因素 | 控制C可以阻断混淆路径 |
| **对撞结构** | A→B←C | B是对撞变量 | 不要控制B，否则会引入偏差 |

## 3. Do演算 (Do-Calculus)

### 3.1 干预算子

`do(T=t)`：将变量T的值强行设置为t，而不是观察到的值
- 观察分布：P(Y|T=t)：给定T=t时Y的条件分布
- 干预分布：P(Y|do(T=t))：当所有人都接受处理t时Y的分布

### 3.2 三大公理

| 公理 | 规则 | 说明 |
|------|------|------|
| **规则1（插入/删除观察）** | P(y|do(x), z, w) = P(y|do(x), z) 若 (Y ⊥ W | X,Z) 在G_x中 | 如果W和Y在移除X的出边的图中条件独立，可以删除W |
| **规则2（交换行动/观察）** | P(y|do(x), do(z), w) = P(y|do(x), z, w) 若 (Y ⊥ Z | X, W) 在G_{x,z̄}中 | 如果Z和Y在移除X出边和Z入边的图中条件独立，可以将do(z)替换为观察z |
| **规则3（插入/删除行动）** | P(y|do(x), do(z), w) = P(y|do(x), w) 若 (Y ⊥ Z | X, W) 在G_{x,z(W̄)}中 | 如果Z和Y在移除X出边和Z中不是W祖先的节点的入边的图中条件独立，可以删除do(z) |

## 4. 识别方法

### 4.1 后门准则 (Backdoor Criterion)

**定义**：如果变量集Z满足：
1. Z不包含T的后代
2. Z阻断了所有T和Y之间的后门路径（即从T出发指向T的边的路径）

则有：
```
P(Y|do(T=t)) = Σ_z P(Y|T=t, Z=z) P(Z=z)
```

**例子**：
- T→Y，混淆路径T←U→Y
- Z={U} 满足后门准则
- 控制U就可以得到因果效应

### 4.2 前门准则 (Frontdoor Criterion)

**定义**：如果变量集Z满足：
1. Z阻断了所有T到Y的直接路径
2. 没有后门路径从T到Z
3. 所有Z到Y的后门路径都被T阻断

则可以通过Z来识别因果效应，即使存在未观察到的混杂因素。

### 4.3 工具变量 (Instrumental Variable, IV)

**假设**：
1. 工具变量Z和处理T相关（相关性）
2. Z和Y之间没有直接路径（排他性）
3. Z和U之间没有共同原因（外生性）

**估计**：
```
ATE = Cov(Y,Z) / Cov(T,Z)
```

## 5. 中介效应与调节效应

### 5.1 中介效应 (Mediation)

**定义**：处理T通过中介变量M影响结果Y
- 总效应：T→Y的直接效应 + T→M→Y的间接效应
- 自然直接效应 (NDE)：保持M不变，改变T的效应
- 自然间接效应 (NIE)：保持T不变，改变M的效应

**中介分析步骤**：
1. 检验T对Y的总效应显著
2. 检验T对M的效应显著
3. 检验M对Y的效应显著
4. 比较加入M后T的效应是否变小

### 5.2 调节效应 (Moderation)

**定义**：变量W影响T对Y的效应大小
- 模型：Y = a*T + b*W + c*T*W + ε
- 交互项c显著则说明存在调节效应

**异质性处理效应 (HTE)**：
- 不同子群体的处理效应不同
- 可以用因果树、因果森林来识别异质性

## 6. 常见偏差

| 偏差 | 原因 | 避免方法 |
|------|------|----------|
| **选择偏差** | 处理分配不是随机的 | 随机实验、匹配、加权 |
| **混杂偏差** | 存在同时影响T和Y的变量 | 控制混杂变量、后门调整 |
| **对撞偏差** | 控制了对撞变量 | 不要控制对撞变量 |
| **幸存者偏差** | 只观察到了幸存者 | 收集完整样本 |
| **共同方法偏差** | 变量都来自同一测量方法 | 不同数据源交叉验证 |
