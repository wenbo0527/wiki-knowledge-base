# 专题08_AI_Agent_Design - 搜索任务

## 专题背景

AI Agent（人工智能代理）是当前大模型应用领域的核心技术方向。与传统的LLM调用不同，Agent强调自主规划、工具使用、多步骤任务执行和持续学习能力。本专题旨在系统性地收集AI Agent Design的核心知识，包括Skill设计、Agent架构、模型调优和RAG技术等关键领域。
