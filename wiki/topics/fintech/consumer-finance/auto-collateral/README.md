## 核心定义

### 产品类型

| 类型 | 押证 | 押车 | GPS | 说明 |
|------|------|------|-----|------|
| **抵押贷款** | ✅ | ❌ | ❌ | 仅抵押机动车登记证 |
| **GPS贷款** | ❌ | ✅ | ✅ | 安装GPS，押车不押证 |
| **质押贷款** | ✅ | ✅ | ❌ | 车辆交付保管 |

### 车抵贷 vs 传统信贷

| 维度 | 车抵贷 | 传统信贷 |
|------|--------|----------|
| 抵押物 | 汽车 | 无 |
| 额度 | 评估价50-80% | 信用评估 |
| 审批速度 | 快 | 慢 |
| 利率 | 较低 | 较高 |
| 逾期处置 | 可拖车 | 法律诉讼 |

## 商业模式

### 收入结构

| 收入来源 | 说明 |
|----------|------|
| **利息收入** | 主要收入 |
| **GPS服务费** | 设备和服务费 |
| **逾期拖车费** | 逾期处置收入 |
| **车辆处置残值** | 处置抵押物收入 |

### 成本结构

| 成本项 | 说明 |
|--------|------|
| 资金成本 | 获取资金成本 |
| 坏账成本 | 无法回收的损失 |
| GPS成本 | 设备折旧+通信费 |
| 线下成本 | 验车、过户等 |

### 定价模型

```
可贷金额 = 车辆评估价 × 成数

典型成数:
- 新车: 70-80%
- 二手车(3年内): 60-70%
- 二手车(3-5年): 50-60%
- 二手车(5年以上): 40-50%

利率范围:
- 银行系: 8-15%
- 汽车金融公司: 10-18%
- 融资租赁公司: 15-24%
- 小贷/担保公司: 18-36%
```

## 风控体系

### 核心风控流程

```
验车 → 评估 → 查档 → 抵押/安装GPS → 放款 → 贷后监控 → 处置
```

### 风控维度

| 维度 | 数据来源 | 作用 |
|------|----------|------|
| 车辆评估 | 第三方估价 | 确定可贷金额 |
| 车况检测 | 实地验车 | 排除事故车 |
| 查档 | 车管所 | 核实真实性 |
| 行驶轨迹 | GPS数据 | 贷后监控 |
| 违章查询 | 交管局 | 风险评估 |

### GPS监控

| 监控内容 | 预警规则 |
|----------|----------|
| 实时位置 | 离开常用区域 |
| 轨迹异常 | 频繁出入特定场所 |
| 设备拆除 | GPS信号消失 |
| 区域禁入 | 进入二抵公司聚集地 |

## 贷后处置

### 逾期处置流程

```
逾期30天 → 催收沟通 → 上门催收 → 拖车 → 通知 → 车辆评估 → 拍卖
```

### 处置方式

| 方式 | 说明 |
|------|------|
| 协议处置 | 与车主协商卖车 |
| 法院诉讼 | 诉讼后执行 |
| 拖车处置 | 直接拖车拍卖 |
| 债权转让 | 转让债权 |

### 残值计算

```
拍卖残值 = 车辆评估价 × (1 - 折旧率)

典型处置周期:
- 协议处置: 7-15天
- 法院诉讼: 3-6个月
- 直接拖车: 1-7天
```
