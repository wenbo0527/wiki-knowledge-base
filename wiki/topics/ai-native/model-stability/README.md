## 专题定位
本专题聚焦**消费金融风控模型的稳定性评估与监控**，覆盖稳定性指标、监控体系、漂移诊断、应急处置全链路，解决"模型上线效果好，一段时间后效果断崖式下降"的核心痛点，是消费金融公司模型团队的必修课。
### 核心价值
1. **避免效果衰减**：及时发现模型漂移，在效果下降前修复，保障模型持续有效
2. **降低坏账损失**：模型效果下降后如果不及时修复，坏账率会持续上升，损失巨大
3. **满足监管要求**：银保监会要求金融机构建立模型监控机制，稳定性是重要考核指标
4. **指导模型迭代**：稳定性监控结果直接指导模型何时需要重新训练，避免盲目迭代
