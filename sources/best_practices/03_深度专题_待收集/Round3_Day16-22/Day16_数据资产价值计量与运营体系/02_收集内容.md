### 1.2 数据资产价值评估方法

#### 1.2.1 成本法（Cost-Based）

```python
class CostBasedValuation:
    """
    基于成本法的数据资产估值
    
    公式: 资产价值 = 获取成本 + 维护成本 - 损耗价值
    """
    
    def calculate(self, data_asset):
        # 获取成本
        acquisition_cost = (
            data_asset.collection_cost +      # 采集成本
            data_asset.processing_cost +       # 处理成本
            data_asset.storage_cost            # 存储成本
        )
        
        # 维护成本（按年摊销）
        annual_maintenance = (
            data_asset.quality_monitor_cost +  # 质量管理
            data_asset.security_cost +         # 安全防护
            data_asset.compliance_cost         # 合规维护
        )
        amortization_years = 5  # 假设5年摊销期
        total_maintenance = annual_maintenance * amortization_years
        
        # 损耗价值（随时间递减）
        depreciation = (
            (data_asset.obsolescence_rate * amortization_years) * 
            (acquisition_cost + total_maintenance)
        )
        
        # 资产价值
        asset_value = (
            acquisition_cost + 
            total_maintenance - 
            depreciation
        )
        
        return {
            'acquisition_cost': acquisition_cost,
            'maintenance_cost': total_maintenance,
            'depreciation': depreciation,
            'asset_value': asset_value,
            'cost_per_use': asset_value / data_asset.usage_count if data_asset.usage_count > 0 else float('inf')
        }
```

**适用场景**：内部资产管理、新数据建设预算申请

#### 1.2.2 收益法（Income-Based）

```python
class IncomeBasedValuation:
    """
    基于收益法的数据资产估值
    
    公式: 资产价值 = Σ(年收益 / (1+折现率)^年)
    """
    
    def calculate(self, data_asset, discount_rate=0.12, years=5):
        total_value = 0
        
        for year in range(1, years + 1):
            # 计算每年的收益贡献
            yearly_benefits = self._calculate_yearly_benefits(
                data_asset, year
            )
            
            # 折现计算
            present_value = yearly_benefits / ((1 + discount_rate) ** year)
            total_value += present_value
        
        return {
            'total_value': total_value,
            'annual_benefits': self._calculate_yearly_benefits(data_asset, 1),
            'roi': (total_value - data_asset.total_investment) / data_asset.total_investment,
            'payback_period': self._calculate_payback_period(data_asset)
        }
    
    def _calculate_yearly_benefits(self, asset, year):
        """
        计算年度收益贡献
        
        包括：
        - 直接收益：数据产品直接带来的收入
        - 效率收益：决策效率提升带来的人工节省
        - 风险收益：风险规避避免的损失
        - 创新收益：新业务机会带来的潜在收益
        """
        direct_benefits = asset.direct_revenue * (1 + asset.revenue_growth) ** (year - 1)
        efficiency_benefits = asset.efficiency_savings / asset.total_investment * asset.investment_cost
        risk_benefits = asset.annual_risk_exposure * asset.risk_reduction_rate
        innovation_benefits = asset.innovation_potential * 0.1 * year  # 假设创新收益逐年释放
        
        return direct_benefits + efficiency_benefits + risk_benefits + innovation_benefits
```

**适用场景**：向管理层证明数据投入价值、投资决策

#### 1.2.3 市场法（Market-Based）

```python
class MarketBasedValuation:
    """
    基于市场法的数据资产估值
    
    通过对比同类数据资产的市场交易价格进行评估
    """
    
    def calculate(self, data_asset, comparable_assets):
        """
        comparable_assets: 类似数据资产的交易案例
        """
        # 选取可比资产
        comparables = self._select_comparables(
            data_asset, 
            comparable_assets,
            criteria=['data_type', 'volume', 'quality', 'freshness']
        )
        
        # 调整系数
        adjustments = {
            'volume_diff': self._adjust_for_volume(data_asset, comparables),
            'quality_diff': self._adjust_for_quality(data_asset, comparables),
            'freshness_diff': self._adjust_for_freshness(data_asset, comparables),
            'uniqueness_diff': self._adjust_for_uniqueness(data_asset, comparables)
        }
        
        # 加权平均估值
        weights = [0.4, 0.3, 0.2, 0.1]  # 权重分配
        adjusted_values = []
        
        for comp in comparables:
            adj_value = comp.transaction_price
            for i, (factor, adj) in enumerate(adjustments.items()):
                adj_value *= (1 + adj[factor] * weights[i])
            adjusted_values.append(adj_value)
        
        estimated_value = sum(adjusted_values) / len(adjusted_values)
        
        return {
            'estimated_value': estimated_value,
            'comparable_count': len(comparables),
            'adjustments': adjustments,
            'confidence': 'high' if len(comparables) >= 3 else 'medium'
        }
```

**适用场景**：数据交易、数据资产证券化

```
┌────────────────────────────────────────────────────────────────┐
│                数据资产ROI计算公式                                │
├────────────────────────────────────────────────────────────────┤
│                                                                  │
│   ROI = (总收益 - 总成本) / 总成本 × 100%                        │
│                                                                  │
│   其中：                                                          │
│   总成本 = 建设成本 + 运维成本 + 管理成本                         │
│   总收益 = 直接收益 + 效率收益 + 风险收益 + 创新收益               │
│                                                                  │
│   收益计算示例：                                                  │
│   • 直接收益 = 数据产品收入 + 数据服务收入                         │
│   • 效率收益 = 取数时间节省 × 人力成本                            │
│   • 风险收益 = 避免的损失金额 × 风险降低率                         │
│   • 创新收益 = 新业务增收 × 数据贡献度                            │
│                                                                  │
└────────────────────────────────────────────────────────────────┘
```
