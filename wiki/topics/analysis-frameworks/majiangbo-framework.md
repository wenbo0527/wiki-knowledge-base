## 核心理念

> 投资的本质是识别趋势，然后在正确的结构中找到最佳的入场节奏。

## 趋势判断

### 1. 趋势识别

| 趋势类型 | 特征 | 持续时间 |
|----------|------|----------|
| **长周期** | 技术革命/人口结构 | 10-20年 |
| **中周期** | 产业升级/政策周期 | 3-5年 |
| **短周期** | 库存/情绪周期 | 6-18月 |

### 2. 趋势阶段判断

```
趋势生命周期
    │
    ├── 萌芽期：概念热，实际慢
    ├── 狂热期：泡沫化，全面涨
    ├── 冷静期：去伪存真，分化
    ├── 成长期：实质增长，龙头胜
    └── 成熟期：稳定增长，慢牛
```

### 3. 趋势驱动力

| 驱动力类型 | 强度 | 说明 |
|------------|------|------|
| **技术创新** | 强 | 颠覆性 |
| **政策支持** | 中 | 可持续 |
| **需求拉动** | 中 | 市场化 |
| **资本驱动** | 弱 | 易泡沫 |

## 结构分析

### 1. 产业链结构

```
产业链分析框架
    │
    ├── 上游：资源/技术
    │     └── 定价权在哪
    │
    ├── 中游：制造/生产
    │     └── 壁垒在哪
    │
    └── 下游：渠道/用户
          └── 话语权在哪
```

### 2. 利益分配结构

| 环节 | 利润占比 | 议价能力 |
|------|----------|----------|
| **垄断环节** | 最高 | 最强 |
| **瓶颈环节** | 高 | 强 |
| **竞争环节** | 低 | 弱 |
| **过剩环节** | 亏损 | 无 |

### 3. 竞争格局演变

| 格局阶段 | 特征 | 投资策略 |
|----------|------|----------|
| **混战期** | 群雄逐鹿 | 观望 |
| **淘汰期** | 弱势出清 | 选龙头 |
| **稳定期** | 格局清晰 | 买龙头 |
| **演变期** | 新玩家入局 | 关注变化 |

## 节奏把控

### 1. 买入节奏

| 时机 | 信号 | 仓位 |
|------|------|------|
| **黄金坑** | 恐慌暴跌+基本面无损 | 重仓 |
| **合理偏低** | 估值合理+催化剂临近 | 中仓 |
| **合理偏高** | 估值合理但无催化 | 轻仓 |
| **泡沫化** | 全面乐观+故事满天飞 | 清仓 |

### 2. 卖出节奏

| 信号 | 含义 | 动作 |
|------|------|------|
| **逻辑破坏** | 核心假设不成立 | 立即卖 |
| **估值泡沫** | 严重高估 | 分批卖 |
| **更好的机会** | 机会成本高 | 换仓 |
| **流动性需求** | 需要现金 | 减仓 |

### 3. 仓位管理

| 市场状态 | 仓位建议 |
|----------|----------|
| **趋势向上** | 8-9成 |
| **震荡** | 5-7成 |
| **趋势向下** | 3-5成 |
| **恐慌底部** | 8-9成+杠杆 |

## 实战应用

### 案例：分析AI算力机会

| 维度 | 分析 |
|------|------|
| **趋势** | AI长周期趋势明确，当前处于成长期 |
| **结构** | 上游英伟达垄断，中游分化，下游应用爆发 |
| **节奏** | 泡沫化后调整，估值合理后布局 |

### 案例：分析新能源车

| 维度 | 分析 |
|------|------|
| **趋势** | 渗透率进入S曲线加速期 |
| **结构** | 电池占成本40%+，格局初定 |
| **节奏** | 周期底部，龙头布局 |
