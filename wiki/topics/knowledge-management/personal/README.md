## 第二大脑理念

**核心思想**: 把大脑作为思考器官，把外部系统作为存储器官。

```
传统模式
    大脑 → 思考 + 记忆（负担重）
         │
         ▼
    遗忘曲线（记忆衰减）

第二大脑模式
    大脑 → 思考（专注创造）
    外部系统 → 存储（可靠保存）
```

## 主流方法论

### PARA方法

| 类型 | 说明 | 示例 |
|------|------|------|
| **Projects** | 当前在做的项目 | 产品规划、技术方案 |
| **Areas** | 长期负责的领域 | 技术架构、职业发展 |
| **Resources** | 感兴趣的主题 | AI、心理学、写作 |
| **Archives** | 完成后归档 | 旧项目、已结束领域 |

### Zettelkasten（卡片盒）

| 概念 | 说明 |
|------|------|
| **Atomic Notes** | 原子化笔记，一个笔记一个想法 |
| **链接** | 笔记之间通过链接连接 |
| **涌现** | 笔记多了自然会形成网络 |

### LYT框架

| 类型 | 说明 |
|------|------|
| **Daily Notes** | 每日记录 |
| **Fleeting Notes** | 临时想法 |
| **Literature Notes** | 阅读笔记 |
| **Permanent Notes** | 永久笔记 |

## 工作流程

```
输入 → 存储 → 连接 → 输出

输入
├── 阅读（书、文章、播客）
├── 观察（生活、工作）
├── 交流（对话、会议）
└── 思考（灵感、想法）

存储
├── 快速捕获（闪念笔记）
├── 整理归类（永久笔记）
└── 标签系统（元数据）

连接
├── 建立链接
├── 发现关联
└── 形成网络

输出
├── 写作
├── 分享
└── 实践
```
