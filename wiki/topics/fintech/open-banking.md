## API能力矩阵

### 按业务领域分类

| 业务 | API类别 | 典型API | 应用场景 |
|------|---------|---------|----------|
| **账户** | 查询 | 余额查询、交易明细 | 财务管理 |
| | 管理 | 开户、销户、冻结 | 远程服务 |
| **支付** | 付款 | 单笔/批量付款 | 代发工资 |
| | 收款 | 快捷支付、扫码支付 | 电商收款 |
| **信贷** | 申请 | 额度查询、贷款申请 | 消费贷 |
| | 还款 | 还款计划、提前还款 | 贷后管理 |
| **数据** | 信用 | 征信查询、评分服务 | 风控决策 |
| | 洞察 | 消费分析、经营分析 | 精准营销 |

## 国际标准对比

| 标准 | 地区 | 核心特点 |
|------|------|----------|
| **OBIE** | 英国 | 强制开放，9大银行，账户+支付 |
| **Berlin Group** | 欧盟 | PSD2合规，强客户认证(SCA) |
| **FDX** | 美国 | 自愿参与，数据共享导向 |
| **CMA9** | 澳大利亚 | 强制开放，四大银行先行 |
| **JFSA** | 日本 | 银行主导，分阶段开放 |
