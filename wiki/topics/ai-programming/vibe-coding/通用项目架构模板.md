## 一、标准化项目目录结构

### 基础结构（所有项目适用）

```
project/
├── .gitignore                    # Git忽略文件
├── .env                          # 环境变量（不上传）
├── .env.example                 # 环境变量模板
├── README.md                     # 项目说明
├── LICENSE                       # 许可证
├── CLAUDE.md                     # AI行为准则 ⭐ Vibe Coding核心
├── AGENTS.md                     # Agent配置
├── Makefile                      # 自动化脚本
├── requirements.txt              # Python依赖
├── package.json                  # Node依赖
├── docker-compose.yml            # Docker配置
├── Dockerfile                    # Docker镜像
│
├── src/                         # 源代码 ⭐
│   ├── __init__.py
│   ├── main.py                  # 入口文件
│   ├── config/                  # 配置模块
│   ├── models/                  # 数据模型
│   ├── services/               # 业务逻辑
│   ├── api/                    # API接口
│   ├── utils/                  # 工具函数
│   └── core/                   # 核心功能
│
├── tests/                       # 测试代码 ⭐
│   ├── __init__.py
│   ├── unit/                   # 单元测试
│   ├── integration/            # 集成测试
│   └── fixtures/               # 测试数据
│
├── docs/                        # 文档
│   ├── api/                    # API文档
│   ├── guides/                 # 使用指南
│   └── architecture/           # 架构文档
│
├── scripts/                     # 脚本
│   ├── init.sh                 # 初始化脚本
│   ├── deploy.sh               # 部署脚本
│   └── backup.sh               # 备份脚本
│
├── memory-bank/                 # 记忆库 ⭐ Vibe Coding核心
│   ├── @design-doc.md         # 设计文档（PRD/GDD）
│   ├── @tech-stack.md         # 技术栈说明
│   ├── @architecture.md       # 架构说明
│   ├── @implementation-plan.md # 实施计划
│   └── @progress.md           # 进度记录
│
├── notebooks/                   # Jupyter笔记本
├── data/                        # 数据文件
│   ├── raw/                   # 原始数据
│   └── processed/             # 处理后数据
│
└── logs/                        # 日志文件
```

## 五、环境配置模板

### Python .env.example

```bash
# 数据库
DATABASE_URL=postgresql://user:pass@localhost:5432/dbname

# Redis
REDIS_URL=redis://localhost:6379/0

# JWT
JWT_SECRET=your-secret-key
JWT_ALGORITHM=HS256

# API Keys
OPENAI_API_KEY=sk-xxx

# 环境
DEBUG=True
LOG_LEVEL=DEBUG
```

### Docker Compose 模板

```yaml
version: '3.8'

services:
  app:
    build: .
    ports:
      - "8000:8000"
    environment:
      - DATABASE_URL=${DATABASE_URL}
      - REDIS_URL=${REDIS_URL}
    depends_on:
      - db
      - redis
  
  db:
    image: postgres:15
    volumes:
      - postgres_data:/var/lib/postgresql/data
    environment:
      - POSTGRES_USER=user
      - POSTGRES_PASSWORD=pass
      - POSTGRES_DB=dbname
  
  redis:
    image: redis:7-alpine
    volumes:
      - redis_data:/data

volumes:
  postgres_data:
  redis_data:
```
