## 核心内容

### 本体论核心概念

| 概念 | 定义 | 关键要素 |
|:---|:---|:---|
| **Ontology** | 共享概念化的显式规范 | 类、属性、关系、实例 |
| **知识图谱** | 结构化的语义知识库 | 实体、关系、属性 |
| **语义推理** | 基于本体的逻辑推导 | 规则引擎、推理算法 |

### 本体工程方法论

| 阶段 | 任务 | 产出 |
|:---|:---|:---|
| **需求分析** | 领域分析、用例定义 | 需求规格说明 |
| **概念建模** | 类/属性/关系定义 | 概念模型 |
| **形式化** | 本体编码、约束定义 | 形式化本体 |
| **验证** | 一致性检查、推理测试 | 验证报告 |
