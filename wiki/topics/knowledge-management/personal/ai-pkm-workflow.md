### 4. 输出生成

| 场景 | AI能力 | 例子 |

## 工作流集成

### 方案1：AI作为助手

```
笔记工作流 + AI助手

写作 → AI润色 → 人工审核 → 发布
        ↑
        └──── AI可以帮你：
              - 检查语法
              - 优化表达
              - 补充内容
```

### 方案2：AI作为研究伙伴

```
研究工作流 + AI

1. AI帮你读：大量文献 → 核心观点总结
2. AI帮你想：观点碰撞 → 新的洞察
3. AI帮你写：洞察 → 初稿
4. 人工审核 → 发布
```

### 方案3：AI驱动的知识库

```
知识库 + RAG

用户问题
    │
    ▼
RAG系统：召回相关笔记
    │
    ▼
AI基于笔记内容生成答案
    │
    ▼
答案 + 参考来源
```

## 最佳实践

### DO

1. **AI是助手，不是替代** — 核心思考还是靠自己
2. **保持批判思维** — AI说的不一定对
3. **人机协作** — AI做重复性工作，人做创造性工作
4. **持续优化** — 根据效果调整AI使用方式

### DON'T

1. **不要完全依赖AI** — AI会犯错，需要审核
2. **不要忽视基础** — 好的笔记习惯比AI重要
3. **不要过度使用** — AI是工具，不是目的
4. **不要忘记隐私** — 敏感内容慎用AI
