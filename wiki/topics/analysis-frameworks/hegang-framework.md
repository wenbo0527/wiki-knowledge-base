## 第一章：价值投资分析方法论

### 财富齿轮模型

> 理性→价值→投资→人生，四个齿轮环环相扣

```
┌─────────────────────────────────────────────────────────────┐
│                    财富齿轮模型                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│         以理性寻找价值                                        │
│              ⚙️                                              │
│              ↓                                               │
│         以价值指引投资                                        │
│              ⚙️                                              │
│              ↓                                               │
│         以投资赢得人生                                        │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 财富三来源

| 类型 | 说明 | 案例 |
|------|------|------|
| **资源开发型** | 自然资源开采 | 中东石油 |
| **产品生产型** | 制造业 | 中国制造 |
| **服务型** | 劳动/智力商品化 | 芯片设计、在线教育 |

### 财富三流动

| 类型 | 说明 |
|------|------|
| **结构性流动** | 技术升级、品牌建设提升价值 |
| **空间性流动** | 需求驱动财富流向 |
| **时间性流动** | 复利效应 |

### 价值三象限

| 维度 | 判断标准 |
|------|----------|
| **需求端** | 规模足够大？增速足够快？ |
| **价值链** | 在价值链中的位置？定价权？ |
| **长期价值** | 护城河？持续竞争优势？ |

## 第二章：行业价值分析方法论

### 三个维度

```
行业分析三维度
    │
    ├── 第一维度：行业规模
    │     ├── 十亿级/百亿级：投资价值有限
    │     ├── 千亿级：具备投资价值
    │     └── 万亿级：国民经济支柱
    │
    ├── 第二维度：产业链条
    │     ├── 上游主导型：石油化工、芯片
    │     ├── 中游主导型：家电（品牌技术）
    │     └── 下游主导型：咖啡（渠道品牌）
    │
    └── 第三维度：竞争格局
          ├── 垄断型：龙头稳定
          └── 充分竞争：选龙头或搅局者
```

### 行业规模判断

| 级别 | 规模 | 投资价值 |
|------|------|----------|
| 十亿级 | <100亿 | 有限 |
| 百亿级 | 100-1000亿 | 有限 |
| 千亿级 | 1000-10000亿 | 良好 |
| 万亿级 | >10000亿 | 优秀 |

### 产业链条分析

| 类型 | 代表行业 | 投资重点 |
|------|----------|----------|
| **上游主导** | 石油化工、芯片 | 上游龙头 |
| **中游主导** | 家电 | 品牌技术龙头 |
| **下游主导** | 咖啡、餐饮 | 渠道品牌龙头 |

### 竞争格局分析

| 类型 | 特点 | 投资策略 |
|------|------|----------|
| **垄断型** | 三巨头瓜分 | 投龙头企业 |
| **充分竞争** | 高度分散 | 投龙头或有独特优势的搅局者 |

## 第三章：公司价值分析方法论

### 申万行业分类（31个一级行业）

| 大类 | 包含 |
|------|------|
| **大消费** | 农林牧渔、食品饮料、家电、医药、服务类 |
| **周期性行业** | 有色金属、煤炭、钢铁、汽车、房地产 |
| **科技** | 电子、通信、计算机、传媒 |
| **金融** | 银行、非银金融 |
| **其他** | 综合性行业 |

### 公司分析三维度

| 维度 | 分析问题 |
|------|----------|
| **行业占比** | 占全行业收入5%+？行业龙头？ |
| **价值链位置** | 主导链条？非主导？ |
| **竞争地位** | 龙头/寡头/跟随者？ |

## 第四章：基金分析方法论

### 基金投资三步法

```
┌─────────────────────────────────────────────────────────────┐
│                  基金投资三步法                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  Step 1: 选赛道                                             │
│  └── 行业主题/宽基指数                                      │
│                                                              │
│  Step 2: 选基金                                             │
│  └── 历史业绩/基金经理/规模/费率                            │
│                                                              │
│  Step 3: 定配置                                             │
│  └── 核心-卫星策略/定期再平衡                              │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 核心-卫星配置

| 类型 | 比例 | 说明 |
|------|------|------|
| **核心资产** | 60-80% | 指数基金、蓝筹股、债券 |
| **卫星资产** | 20-40% | 成长股、行业主题基金 |

## 第六章：黄金价格分析方法论

### 黄金价格四因素

| 因素 | 说明 |
|------|------|
| **美元走势** | 美元强则黄金弱 |
| **实际利率** | 利率低则黄金强 |
