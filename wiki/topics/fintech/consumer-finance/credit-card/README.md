## 核心定义

### 产品类型

| 类型 | 特点 | 代表产品 |
|------|------|----------|
| **贷记卡** | 循环信用，有免息期，最长50+天 | 各银行信用卡 |
| **准贷记卡** | 先存后用，有透支功能 | 少数银行产品 |
| **借记卡** | 先存后花，不可透支 | 预付卡、礼品卡 |
| **虚拟卡** | 纯线上，无实体卡 | 云闪付、虚拟信用卡 |

### 信用卡vs消费分期

| 维度 | 信用卡 | 消费分期 |
|------|--------|----------|
| 额度 | 循环额度 | 单次额度 |
| 免息期 | 有 | 无 |
| 期限 | 灵活 | 固定3/6/12期 |
| 场景 | 任意 | 指定场景 |

## 商业模式

### 收入结构

| 收入来源 | 说明 | 占比 |
|----------|------|------|
| **利息收入** | 循环信用利息 | ~60% |
| **手续费** | 分期手续费、预借现金 | ~20% |
| **商户回佣** | MCC手续费分成 | ~15% |
| **年费** | 高端卡年费收入 | ~5% |

### 成本结构

| 成本项 | 说明 |
|--------|------|
| 资金成本 | 吸储或同业资金成本 |
| 坏账成本 | 不良贷款损失 |
| 营销成本 | 获客、活客费用 |
| 运营成本 | 风控、客服、支付清算 |

## 核心能力

### 1. 支付清算

```
发卡行 → 清算网络(银联/Visa) → 收单行 → 商户
  ↑                                      ↓
  ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ← ←
```

### 2. 信用管理

| 能力 | 说明 |
|------|------|
| 额度管理 | 动态调整信用额度 |
| 欺诈检测 | 异常交易识别 |
| 催收管理 | 逾期资产处置 |

### 3. 权益运营

| 权益类型 | 说明 |
|----------|------|
| 机场贵宾 | 机场CIP、安检口贵宾 |
| 酒店权益 | 高端酒店折扣/升级 |
| 加油返现 | 加油返现比例 |
| 购物优惠 | 联名卡专属折扣 |
