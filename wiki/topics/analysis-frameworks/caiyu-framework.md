## 核心理念

> 分析一个生意的本质，要从商业逻辑出发，理解用户价值主张，然后看产品策略如何支撑。

### 三层分析模型

```
┌─────────────────────────────────────────────────────────────┐
│                    蔡钰商业分析三层                             │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  第一层：商业逻辑                                             │
│  ├── 怎么赚钱（收入模式）                                    │
│  ├── 为什么能赚钱（壁垒）                                    │
│  └── 能赚多久（持续性）                                      │
│                                                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  第二层：用户价值                                             │
│  ├── 用户是谁（细分群体）                                    │
│  ├── 解决什么问题（痛点）                                    │
│  └── 值不值（价值感知）                                     │
│                                                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  第三层：产品策略                                             │
│  ├── 做什么产品（范围）                                      │
│  ├── 怎么做（体验设计）                                      │
│  └── 怎么卖（渠道策略）                                      │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 商业逻辑层

### 1. 收入模式识别

| 收入模式 | 特征 | 典型案例 |
|----------|------|----------|
| **一次性卖货** | 卖货赚差价 | 传统零售 |
| **订阅制** | 持续收费 | Netflix、会员 |
| **平台抽成** | 撮合交易收佣金 | 淘宝、美团 |
| **广告** | 流量变现 | 百度、抖音 |
| **增值服务** | 基础免费+高级付费 | Freemuem |
| **授权费** | 知识产权变现 | 专利、IP |

### 2. 壁垒分析

| 壁垒类型 | 强度 | 说明 |
|----------|------|------|
| **规模效应** | 强 | 成本随规模下降 |
| **网络效应** | 很强 | 用户越多越值钱 |
| **转换成本** | 强 | 用户换产品代价大 |
| **品牌** | 中 | 需要持续投入 |
| **成本优势** | 中 | 地理位置、资源禀赋 |

### 3. 商业模式画布

| 要素 | 分析问题 |
|------|----------|
| **客户细分** | 谁是核心客户 |
| **价值主张** | 解决什么问题 |
| **渠道** | 如何触达客户 |
| **客户关系** | 如何维护 |
| **收入来源** | 如何收钱 |
| **核心资源** | 什么是壁垒 |
| **关键活动** | 必须做什么 |
| **合作伙伴** | 谁是盟友 |

## 用户价值层

### 1. 用户细分

```
用户分层模型
    │
    ├── 超级用户（1%）：贡献50%收入
    │     └── 重点服务、口碑传播
    │
    ├── 普通用户（19%）：贡献30%收入
    │     └── 激活转化、提升ARPU
    │
    └── 潜在用户（80%）：贡献20%收入
          └── 拉新、激活
```

### 2. 用户旅程分析

| 阶段 | 用户行为 | 分析重点 |
|------|----------|----------|
| **认知** | 知道产品 | 渠道触达 |
| **兴趣** | 主动了解 | 内容营销 |
| **欲望** | 想要试试 | 价值展示 |
| **行动** | 开始使用 | 转化率 |
| **留存** | 持续使用 | LTV |
| **传播** | 推荐他人 | NPS |

### 3. 价值感知评估

| 维度 | 问题 |
|------|------|
| **功能价值** | 解决问题了吗 |
| **情感价值** | 使用体验愉悦吗 |
| **社交价值** | 能炫耀吗 |
| **价格感知** | 值这个价吗 |

## 产品策略层

### 1. 产品定位

| 定位类型 | 说明 | 案例 |
|----------|------|------|
| **市场补缺者** | 细分空白市场 | 垂直电商 |
| **挑战者** | 正面竞争 | 拼多多vs京东 |
| **领导者** | 主导市场 | 微信 |
| **追随者** | 模仿复制 | 众多模仿者 |

### 2. 产品矩阵

```
产品矩阵分析
    │
    ├── 流量型产品（引流）
    │     └── 低毛利/免费/高频
    │
    ├── 利润型产品（赚钱）
    │     └── 高毛利/付费/中频
    │
    └── 战略型产品（壁垒）
          └── 亏损/战略投入
```

### 3. 定价策略

| 策略 | 适用场景 | 案例 |
|------|----------|------|
| **渗透定价** | 抢占市场 | 小米早期 |
| **价值定价** | 差异化 | 苹果 |
| **撇脂定价** | 早期收割 | 新品首发 |
| **动态定价** |供需调节 | 滴滴 |

## 实战应用

### 案例：分析瑞幸咖啡

| 层次 | 分析 |
|------|------|
| **商业逻辑** | 平台抽成+自营，规模化后盈利 |
| **用户价值** | 低价+便利+社交货币 |
| **产品策略** | 爆品策略+私域运营 |

### 案例：分析得到

| 层次 | 分析 |
|------|------|
| **商业逻辑** | 知识付费订阅制 |
| **用户价值** | 认知提升+知识焦虑缓解 |
| **产品策略** | 头部IP+专栏+电子书 |
