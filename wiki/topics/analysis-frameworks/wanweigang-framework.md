```
┌─────────────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1. 实验验证                                               │
```

## 科学思维工具箱

### 1. 物理学思维

| 模型 | 物理来源 | 应用场景 |
|------|----------|----------|
| **能量守恒** | 物理定律 | 资源分配、投入产出 |
| **熵增定律** | 热力学 | 系统演化、效率降低 |
| **临界点** | 相变 | 市场突破、转折点 |
| **非线性** | 物理 | 复利、指数增长 |

### 2. 经济学思维

| 模型 | 核心观点 | 应用 |
|------|----------|-----|
| **机会成本** | 有所得必有所失 | 决策权衡 |
| **激励分析** | 人对激励的反应 | 预测行为 |
| **博弈论** | 策略互动 | 竞争策略 |
| **边际效应** | 最后一个单位的价值 | 优化决策 |

### 3. 心理学思维

| 偏差 | 说明 | 决策影响 |
|------|------|----------|
| **确认偏误** | 只信自己对的 | 信息筛选 |
| **损失厌恶** | 失去比得到痛苦 | 风险决策 |
| **锚定效应** | 第一印象决定判断 | 定价/谈判 |
| **光环效应** | 以偏概全 | 人事决策 |

## 高手成长路径

### 刻意练习五要素

```
刻意练习
    │
    ├── 1. 明确目标：具体可衡量
    │     └── "我想提升" → "我要掌握XX"
    │
    ├── 2. 专注投入：全神贯注
    │     └── 排除干扰，高度专注
    │
    ├── 3. 及时反馈：知道对错
    │     └── 导师/数据/自我复盘
    │
    ├── 4. 舒适区外：适度挑战
    │     └── 太容易无聊，太难焦虑
    │
    └── 5. 大量重复：持续迭代
          └── 10000小时定律
```

## 决策框架

### 不确定性决策

| 工具 | 说明 | 适用场景 |
|------|------|----------|
| **贝叶斯更新** | 根据新证据调整判断 | 概率判断 |
| **小样本验证** | 先小规模测试 | 市场验证 |
| **反向推理** | 如果A则B，反之？ | 逻辑检验 |
| **二阶思维** | 思考后果的后果 | 长期规划 |

### 信息处理

```
信息处理流程
    │
    ├── 收集：多源信息，避免偏食
    │
    ├── 验证：交叉检验，不轻信
    │
    ├── 整合：构建完整图景
    │
    └── 输出：形成可行动结论
```

## 跨界迁移方法论

### 万维钢式学习

```
跨界迁移
    │
    ├── Step 1: 深潜本源
    │     └── 彻底理解原领域模型
    │
    ├── Step 2: 提取本质
    │     └── 抽象核心逻辑/原理
    │
    ├── Step 3: 寻找类比
    │     └── 在新领域找相似结构
    │
    └── Step 4: 验证边界
          └── 检验迁移的适用范围
```

### 经典迁移案例

| 原领域 | 核心原理 | 迁移目标 |
|--------|----------|----------|
| 物理 | 临界点/相变 | 市场突破 |
| 生物 | 进化论/自然选择 | 商业竞争 |
| 物理 | 能量守恒 | 资源分配 |
| 心理 | 认知偏差 | 投资决策 |

## 投资应用

### 万维钢vs何刚框架对比

| 维度 | 万维钢 | 何刚 |
|------|--------|------|
| **核心** | 思维工具 | 投资分析 |
| **方法** | 科学验证 | 宏观+行业+公司 |
| **优势** | 防止认知偏差 | 系统化分析 |

### 组合使用

```
投资决策
    │
    ├── 用万维钢 → 避免认知偏差
    │     ├── 损失厌恶 → 避免频繁交易
    │     ├── 锚定效应 → 不被价格锚定
    │     └── 确认偏误 → 主动寻找反证
    │
    └── 用何刚 → 系统化分析
          ├── 宏观 → 判断时机
          ├── 行业 → 选择赛道
          └── 公司 → 精选标的
```
