## 核心洞察

### Vibe Coding定义

> "用自然语言驱动开发，先跑起来再迭代优化" —— 核心是速度，不是完美

### 适用场景

| 适合 ✅ | 不适合 ⚠️ |
|--------|-----------|
| 快速原型验证 | 复杂业务逻辑 |
| 小工具/脚本 | 高并发系统 |
| 代码片段生成 | 安全敏感代码 |
| 学习新技术 | 大型团队项目 |

## 方法论

### 核心流程

1. **需求描述** → 自然语言描述
2. **AI生成** → 生成初始代码
3. **快速验证** → 立即运行
4. **迭代优化** → 小步快跑

### Prompt技巧

| 技巧 | 示例 |
|------|------|
| 角色设定 | "你是Python专家，熟悉FastAPI..." |
| 输出格式 | "用TypeScript，包含类型定义" |
| 约束条件 | "不依赖外部API，使用本地存储" |
