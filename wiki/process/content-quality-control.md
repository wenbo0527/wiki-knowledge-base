# 内容输出品控手册

> 基于《得到品控手册》改编，适用于个人创作品控
> 
> **适用范围**：知乎问答、知乎文章、公众号文章、Wiki知识库内容
> 
> **更新日期**：2026-04-10

---

## 一、事实核对维度：确保内容准确可溯源

### 核心信息校验

- [ ] **人物**：姓名、头衔、身份信息准确，避免张冠李戴
- [ ] **时间**：事件发生、发布、生效时间精准
- [ ] **地点**：涉及机构、场地、地域名称准确
- [ ] **事物**：产品、概念、术语表述对应正确
- [ ] **数字**：数据来源清晰，百分比、金额、排名等数值准确，引用数据需标注出处

### 专业内容校验

- [ ] 专业领域内容符合领域共识，学术观点、行业术语使用准确
- [ ] 引用案例经过核实，避免脑补细节或套用不相关案例
- [ ] 跨学科内容需确认边界领域规则，避免混淆专业分界

### 特殊类型内容校验

- [ ] 书籍核心观点还原准确，不曲解作者原意
- [ ] 学术类内容是否做到"翻译式转化"：将专业问题匹配现实问题，转化语言表述
- [ ] 历史/文学类内容是否做了事实复核，如历史事件时间线、人物生平细节

---

## 二、合规要求维度：筑牢内容生产底线

### 法律合规

- [ ] 符合国家法律法规，无违反《网络安全法》《著作权法》等条款内容
- [ ] 宣传内容未使用极限词或误导性措辞（如"最棒""第一"等违规表述）
- [ ] 使用的第三方素材（图片、音频、文字片段）均获合法授权
- [ ] 符合重大选题备案要求（如适用）

### 内容红线禁区

- [ ] 未造假或美化数据，如虚构用户评价、夸大效果
- [ ] 未发布无意义的噪音内容（如无价值的冗余信息）
- [ ] 未使用规范性问题之外的极端/煽动性表述
- [ ] 未出现针对民族、宗教、性别、地域等敏感群体的不当表述

### 敏感内容处理

- [ ] 涉及负面事件报道不过度渲染、不做恶意经济判断
- [ ] 涉及争议性话题保持中立，不迎合情绪、不煽动对立
- [ ] 个人隐私信息已做脱敏处理

---

## 三、风格表达维度：确保内容质量

### 表达准确性

- [ ] 核心观点表达准确：使用"不是而是法"纠正认知偏差，或直击核心
- [ ] 无残缺知识输出，解释知识点时补充必要背景和逻辑链
- [ ] 避免空泛表述，用具体案例和感性素材支撑观点

### 认知表达

- [ ] 认知迁移过程清晰：明确展示从原有认知到新知识的过渡
- [ ] 左手挥法/不是而是法等表达技巧使用恰当

### 品牌调性

- [ ] 传递"专业可信、积极进取"的个人品牌调性
- [ ] 风格贴合内容类型：专业类严谨易懂、故事类有代入感
- [ ] 体现"人际互动感"：有明确行动建议、使用适当情绪词

---

## 四、AI协作品控维度：确保AI辅助创作质量

### AI生成内容校验

- [ ] AI生成内容已进行人工复核，非直接发布
- [ ] AI提供的事实性陈述已交叉验证
- [ ] AI引用的数据/案例已标注来源

### AI协作质量检查

- [ ] 核心观点来自本人思考，AI仅辅助表达和整理
- [ ] 个人经验和案例已核实，AI未混入未经确认的信息
- [ ] 文章的个人风格和独特视角保持一致
- [ ] AI生成草稿后，已进行"去AI味"修改

### 知识增量验证

- [ ] 文章具备知识增量，不是已有信息的简单搬运
- [ ] 贴合读者需求，解决真实学习/工作痛点
- [ ] 核心认知交付点清晰，包含"认知起点-迁移过程-终点"逻辑

### 发布前最终检查

- [ ] 全文错别字、语病已排查修改
- [ ] 标题吸引且准确，避免标题党
- [ ] 配图/素材来源合规，已标注引用

---

## 五、质量交付维度：确保内容完成度

### 自我审核清单

发布前快速过一遍：

**事实层**
- [ ] 核心信息（人物/时间/数字）是否准确？
- [ ] 引用数据是否有来源标注？

**合规层**
- [ ] 是否有敏感表述？
- [ ] 是否有夸大/虚假内容？

**质量层**
- [ ] 是否有知识增量？
- [ ] 是否解决了某个真实问题？
- [ ] 个人风格是否体现？

**AI协作层**
- [ ] AI生成内容是否已复核？
- [ ] 是否保持了个人独特视角？

## 使用说明

### 日常创作流程

```
创作初稿
    │
    ▼
AI辅助优化（如适用）
    │
    ▼
品控手册自查
    │
    ▼
发布
```

### Wiki内容创作

Wiki内容相比公众号文章：
- 更侧重**知识增量**验证
- 更侧重**关联性**（与现有内容的链接）
- 格式要求较低，重点在内容质量

### 知乎/公众号文章

相比Wiki内容：
- 更侧重**表达风格**和**读者体验**
- 更侧重**标题吸引力**
- 需要关注**传播性**
