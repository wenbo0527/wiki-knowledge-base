## 一、SWOT分析

### 适用场景
- 战略规划
- 竞争分析
- 项目评估

### 框架

```
┌─────────────────────────────────────┐
│           SWOT 分析                       │
├─────────────────┬─────────────────────┤
│  优势 S        │  劣势 W              │
│  内部积极      │  内部消极            │
├─────────────────┼─────────────────────┤
│  机会 O        │  威胁 T              │
│  外部积极      │  外部消极            │
└─────────────────┴─────────────────────┘
```

### 使用技巧
- S+W是内因，O+T是外因
- SO组合要最大化
- WT组合要最小化

## 二、PESTEL分析

### 适用场景
- 宏观环境分析
- 行业政策评估
- 市场准入分析

### 七大维度

| 维度 | 说明 | 案例 |
|------|------|------|
| **P 政治** | 政策、法规、政府关系 | 行业准入 |
| **E 经济** | GDP、利率、汇率、消费 | 市场容量 |
| **S 社会** | 人口、文化、价值观 | 需求变化 |
| **T 技术** | 创新、研发、专利 | 产品迭代 |
| **E 环境** | 气候、环保、ESG | 可持续发展 |
| **L 法律** | 劳动法、反垄断、消费者 | 合规成本 |

## 三、波特五力

### 适用场景
- 行业竞争分析
- 投资评估
- 战略制定

### 五力模型

```
                    供应商议价能力
                          │
                          ▼
        新进入者 ────► 行业内竞争 ◄─── 替代品
                          │
                          ▼
                    买方议价能力
```

### 分析要点

| 五力 | 分析问题 |
|------|----------|
| **行业内竞争** | 集中度、差异化、转换成本 |
| **供应商议价** | 集中度、可替代性 |
| **买方议价** | 集中度、信息透明度 |
| **新进入者** | 壁垒高低、规模经济 |
| **替代品** | 功能替代、价格替代 |

## 四、波士顿矩阵（BCG）

### 适用场景
- 产品组合分析
- 资源分配
- 投资决策

### 四象限

```
                        相对市场占有率
                    高           低
                ┌─────────┬─────────┐
         高     │ 明星     │ 问题    │
    市         │ ★        │ ?       │
    场         ├─────────┼─────────┤
    成         │ 现金牛   │ 瘦狗    │
    长         │ $        │ ×       │
    率         └─────────┴─────────┘
                   
```

### 策略建议

| 象限 | 策略 |
|------|------|
| 明星 | 投资扩张 |
| 问题 | 选择性投资 |
| 现金牛 | 稳定产出 |
| 瘦狗 | 收割/退出 |

## 五、价值链分析

### 适用场景
- 竞争优势识别
- 成本优化
- 战略定位

### 基本活动

```
供应商 ──► 进货物流 ──► 生产运营 ──► 出货物流 ──► 营销销售 ──► 服务 ──► 客户
                     │
              ┌──────┴──────┐
              │   支持活动   │
              │  企业基础设施│
              │    技术开发   │
              │    采购      │
              │    人力资源   │
              └─────────────┘
```

### 分析要点
- 识别关键价值活动
- 寻找成本优势来源
- 发现差异化机会

## 六、波特竞争战略

### 三种基本战略

| 战略 | 核心 | 适用条件 |
|------|------|----------|
| **成本领先** | 做到成本最低 | 价格敏感行业 |
| **差异化** | 独特价值主张 | 消费者愿意付费 |
| **集中化** | 细分市场深耕 | 资源有限 |

### 战略选择矩阵

```
                        竞争优势
                    多         少
                ┌───────┬───────┐
    目标市场   │        │        │
    宽        │ 总成本 │ 差异   │
    (广泛市场) │ 领先   │ 化    │
              ├───────┼───────┤
    窄        │ 成本  │ 集中  │
   (狭窄市场) │ 集中   │ 差异  │
              └───────┴───────┘
```

## 九、RFM模型

### 适用场景
- 用户分层
- 精准营销
- 客户价值评估

### 三个维度

| 维度 | 说明 | 分析价值 |
|------|------|----------|
| **R 最近一次** | 上次购买距今 | 流失风险 |
| **F 购买频次** | 购买次数 | 忠诚度 |
| **M 消费金额** | 总消费额 | 贡献度 |

### 用户分层

| 类型 | RFM特征 | 策略 |
|------|----------|------|
| 重要价值 | 高F高M低R | VIP维护 |
| 重要召回 | 低F高M高R | 唤回 |
| 重要发展 | 高F低M低R | 提升消费 |
| 流失风险 | 低F低M高R | 流失预警 |

## 十、AARRR海盗模型

### 适用场景
- 增长分析
- 运营优化
- 产品迭代

### 五环节

```
获取(Aquisition) ──► 激活(Activation) ──► 留存(Retention)
        │                                       ▲
        │                                       │
        ▼                                       │
      推荐(Referral) ◄── 收入(Revenue) ◄───────┘
```

### 分析指标

| 环节 | 核心指标 |
|------|----------|
| 获取 | CAC、获客渠道 |
| 激活 | 激活率、首单时间 |
| 留存 | 留存率、LTV |
| 收入 | ARPU、付费率 |
| 推荐 | K因子、NPS |
