## 核心问题

因果推断回答的根本问题是：

> **如果我们采取了不同的行动，会发生什么？**

### 从业务语言到因果语言

| 业务问题 | 因果问题 | 关键概念 |
|---------|---------|----------|
| "发优惠券能提升GMV吗？" | 优惠券对GMV的因果效应是多少？ | 平均处理效应(ATE) |
| "逾期用户有什么特征？" | 这些特征是否**导致**逾期？ | 混杂因素 |
| "新功能提升了留存吗？" | 排除自选择偏差后的净效应？ | 选择偏差 |
| "风控策略是否过度保守？" | 策略收紧的边际效应递减点？ | 异质性处理效应 |

## 两大理论框架

### 1. 潜在结果框架 (Rubin Causal Model)

**核心思想**: 因果是反事实的比较

```
对于个体i:
- Yi(1): 接受处理时的潜在结果
- Yi(0): 未接受处理时的潜在结果
- 个体处理效应: τi = Yi(1) - Yi(0)

核心问题: 我们只能观察到 Yi(1) 或 Yi(0) 中的一个！
```

**关键假设**:
1. **SUTVA**: 稳定单位处理值假设 (无溢出效应)
2. **可忽略性**: 给定协变量，处理分配与潜在结果独立
3. **正值性**: 每个个体都有接受/不接受处理的概率

**适合场景**: 
- A/B测试设计与分析
- 随机化实验
- 观察性研究中的匹配/加权

### 2. 结构因果模型 (Pearl's SCM)

**核心思想**: 因果是数据生成过程的结构性特征

```
因果图 (DAG):
    X → Y    : X直接导致Y
    X ← Z → Y: Z是X和Y的混杂因素
    X → M → Y: M是X到Y的中介变量
    
do-演算: P(Y|do(X=x)) - 干预分布
```

**关键能力**:
1. **混杂控制**: 识别最小调整集
2. **中介分析**: 直接效应 vs 间接效应
3. **反事实推理**: "如果当时...会怎样"
4. **因果发现**: 从数据中学习因果结构

**适合场景**:
- 复杂系统的因果建模
- 无法随机化的观察性研究
- 需要回答反事实问题
- 多变量间的因果路径分析

### 框架选择指南

| 问题类型 | 推荐框架 | 理由 |
|---------|---------|------|
| A/B测试分析 | Rubin | 直接、易解释、与实验设计天然契合 |
| 观察性研究(低维) | Rubin | 匹配、倾向得分方法成熟 |
| 复杂多变量系统 | Pearl | 因果图可视化、do-演算强大 |
| 中介分析 | Pearl | 路径分解清晰 |
| 反事实预测 | Pearl | do-演算直接计算反事实 |

**最佳实践**: 两个框架理解互通，实际分析中结合使用。

## 关键方法速查

### 识别策略 (Identification Strategies)

| 方法 | 适用场景 | 核心假设 | 难度 |
|------|---------|---------|------|
| **随机化实验** | 可控制实验环境 | 随机分配 | ⭐ |
| **匹配/重加权** | 观察性研究，低维 | 可忽略性 | ⭐⭐ |
| **工具变量(IV)** | 有有效工具变量 | 相关性+排他性 | ⭐⭐⭐ |
| **双重差分(DiD)** | 政策评估，面板数据 | 平行趋势 | ⭐⭐ |
| **断点回归(RDD)** | 有清晰 cutoff | 局部随机化 | ⭐⭐⭐ |
| **合成控制** | 单一处理单位 | 预处理拟合 | ⭐⭐⭐ |
| **因果图/do-演算** | 复杂系统 | 图结构正确 | ⭐⭐⭐⭐ |

### 金融风控中的典型应用

```
┌─────────────────────────────────────────────────────────────┐
│  场景1: 风控策略效果评估                                       │
│  ─────────────────────                                       │
│  问题: 新策略上线后逾期率下降，是策略有效还是用户质量变好？      │
│  方法: DiD - 对比策略上线前后、实验组vs对照组                   │
│  关键: 确保平行趋势假设，排除宏观环境因素                       │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  场景2: 营销触达归因                                          │
│  ────────────────                                            │
│  问题: 收到营销短信的用户转化率更高，是短信有效还是高意向用户      │
│       更容易收到短信？                                        │
│  方法: 倾向得分匹配(PSM) 或 IV (如果存在工具变量)              │
│  关键: 控制用户历史行为等混杂因素                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│  场景3: 产品功能因果分析                                       │
│  ─────────────────────                                       │
│  问题: 使用"智能推荐"功能的用户留存更高，是推荐有效还是          │
│       高活跃用户更容易发现该功能？                              │
│  方法: 因果图分析混杂路径 + 匹配/回归调整                      │
│  关键: 识别并控制自选择偏差                                    │
└─────────────────────────────────────────────────────────────┘
```

## 推荐学习资源

### 经典书籍

**入门** (选一本)
- 《The Book of Why》- Judea Pearl (因果推断科普)
- 《Causal Inference: The Mixtape》- Scott Cunningham (计量应用)

**进阶**
- 《Causal Inference in Statistics》- Pearl, Glymour, Jewell (因果图理论)
- 《Mostly Harmless Econometrics》- Angrist & Pischke (实证方法)
- 《Causal Inference for Statistics, Social, and Biomedical Sciences》- Imbens & Rubin (潜在结果理论)

### 在线课程

- **Coursera**: "Causal Inference" (Columbia University)
- **EdX**: "Data Analysis for Social Scientists" (MIT)
- **Brady Neal's Course**: "Introduction to Causal Inference" (免费)

- `econml` - Microsoft的计量经济学ML
