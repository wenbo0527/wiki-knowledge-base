## 核心定义

### 产品类型

| 类型 | 额度 | 期限 | 特点 |
|------|------|------|------|
| **小额现金贷** | 500-5000 | 7-30天 | 纯线上，秒批 |
| **薪金贷** | 月薪3-10倍 | 3-36月 | 工资流水授信 |
| **信用贷** | 1-50万 | 12-60月 | 综合信用评估 |
| **场景现金贷** | 1-10万 | 6-24月 | 指定用途 |

### 现金贷 vs 消费分期

| 维度 | 现金贷 | 消费分期 |
|------|--------|----------|
| 到账形式 | 现金到账 | 场景支付 |
| 用途控制 | 无 | 部分场景可控制 |
| 定价 | APR较高 | 费率透明 |
| 期限 | 短 | 较长 |

## 商业模式

### 收入结构

| 收入来源 | 说明 |
|----------|------|
| **利息收入** | 主要收入 |
| **服务费** | 一次性服务费 |
| **贷后服务费** | 逾期催收费 |

### 定价模型

```
年化利率(APR) = 资金成本 + 风险成本 + 运营成本 + 利润

典型现金贷 APR:
- 银行系: 12-24%
- 消金公司: 18-36%
- 互金平台: 24-100%+
```

## 风控体系

### 核心风控流程

```
申请 → 身份验证 → 信用评估 → 额度审批 → 放款 → 贷后管理
```

### 风控维度

| 维度 | 数据来源 | 作用 |
|------|----------|------|
| 身份验证 | 公安、运营商 | 反欺诈 |
| 信用评估 | 央行征信、第三方 | 信用评分 |
| 还款能力 | 银行流水、社保 | 收入核实 |
| 还款意愿 | 社交、设备 | 行为分析 |

### 定价矩阵

| 评分段 | 利率 | 额度 |
|--------|------|------|
| 优质 | 12-18% | 高 |
| 良好 | 18-24% | 中 |
| 一般 | 24-36% | 低 |
| 次级 | 36%+ | 极低 |

## 监管合规

### 重要监管文件

| 政策 | 核心要求 |
|------|----------|
| 141号文 | 现金贷综合费率36%上限 |
| 141号文 | 不得暴力催收 |
| 网络小贷新规 | 跨省经营限制 |
| 个人信息保护法 | 数据合规要求 |

### 合规红线

| 红线 | 说明 |
|------|------|
| 利率上限 | 综合费率36%以内 |
| 暴力催收 | 禁止爆通讯录 |
| 砍头息 | 不得预扣利息 |
| 资质要求 | 无资质不得经营 |
