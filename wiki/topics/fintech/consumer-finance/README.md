## MECE 子分类

### 1. [[consumer-finance/credit-card|信用卡业务]]
**定义**: 向个人消费者提供先消费后还款的信用支付工具

| 产品类型 | 说明 |
|----------|------|
| 贷记卡 | 循环信用，有免息期 |
| 借记卡 | 先存后花，场景预付 |
| 特权卡 | 机场贵宾、高端权益 |

**核心能力**: 支付清算 + 信用管理 + 权益运营

### 2. [[consumer-finance/installment|消费分期]]
**定义**: 基于特定消费场景的分期付款服务

| 场景 | 代表产品 |
|------|----------|
| 医美 | 医美分期 |
| 教育 | 教育分期（教育贷） |
| 3C数码 | 手机、电脑分期 |
| 家电 | 家电分期 |
| 装修 | 装修贷 |

**核心能力**: 场景接入 + 资金垫付 + 担保兜底

### 3. [[consumer-finance/cash-loan|现金贷]]
**定义**: 直接发放现金的信用贷款，有明确还款周期

| 产品类型 | 特点 |
|----------|------|
| 小额现金贷 | 几千-几万，7-30天 |
| 工资贷 | 基于工资流水 |
| 场景现金贷 | 特定用途现金 |

**核心能力**: 风控定价 + 定价能力 + 催收

### 4. [[consumer-finance/assisted-lending|助贷/联合贷款]]
**定义**: 金融机构与平台合作，共同提供信贷服务

| 模式 | 说明 |
|------|------|
| 助贷 | 平台提供流量+风控，金融机构出资 |
| 联合贷款 | 多家机构按比例出资+风控 |
| 导流 | 仅提供客户，不参与风控 |

**核心能力**: 获客 + 风控 + 资金

### 5. [[consumer-finance/auto-collateral|车抵贷]]
**定义**: 以汽车作为抵押物的贷款产品

| 产品类型 | 说明 |
|----------|------|
| 抵押贷款 | 押证不押车 |
| GPS贷款 | 押车不押证 |
| 质押贷款 | 车辆交付保管 |

**核心能力**: 车辆评估 + GPS监控 + 处置能力

    ▼

### 消费金融核心指标

| 指标 | 说明 |
|------|------|
| **不良率(NPL)** | 逾期90+天贷款/总贷款 |
| **核准率** | 审批通过量/申请量 |
| **件均金额** | 平均单笔贷款金额 |
| **期限** | 平均贷款期限 |
| **利率** | APR/IRR |
| **拨备覆盖率** | 准备金/不良贷款 |
